# ContextLingo

Web này được tạo ra để tham dự hackathon Kaggle **Google DeepMind - Vibe Code with Gemini 3 Pro in AI Studio Kaggle Hackathon** bởi nhóm 2 người: mình và Jaxyss.

Xem app trên AI Studio: https://ai.studio/apps/drive/1VoInyJ8dVJUbsa0yKJqXxja7w-cr6HGg

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
